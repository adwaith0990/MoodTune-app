MoodTune is an AI-powered music recommendation web application built using Python, Streamlit, and Pandas.
The system analyzes Spotify song features such as energy, valence, and danceability to automatically classify songs into different moods like:

😊 Happy
😔 Sad
😌 Chill
🏋️ Workout
🎉 Party
